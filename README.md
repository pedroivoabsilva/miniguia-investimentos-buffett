# 📊 Miniguia de Estudo – Filosofia de Investimentos de Warren Buffett

## 📌 Sobre o Projeto

Este repositório apresenta um estudo estruturado sobre os **12 princípios que orientam a filosofia de investimentos de Warren Buffett**.

O material foi produzido utilizando técnicas de **curadoria de conteúdo, engenharia de prompts e síntese de conhecimento com apoio de IA**.

A base do estudo é o livro **O Jeito Warren Buffett de Investir**, escrito por Robert G. Hagstrom, além de conteúdos educacionais em vídeo sobre investimentos e educação financeira.

---

## 🎯 Objetivos do Projeto

Este projeto busca:

- Compreender os **12 princípios de investimento de Warren Buffett**
- Desenvolver visão crítica para **avaliar empresas**
- Entender conceitos fundamentais de **educação financeira**
- Diferenciar **poupança, investimento e reserva de emergência**
- Criar um **miniguia de revisão rápida**

## 📖 Glossário

| Conceito | Definição |
|--------|--------|
| Value Investing | Estratégia de investir em empresas subvalorizadas |
| Margem de Segurança | Comprar ativos abaixo do valor real |
| ROE | Retorno sobre patrimônio |
| Moat | Vantagem competitiva sustentável |
| Liquidez | Facilidade de transformar um ativo em dinheiro |

## 🚀 Conclusão

A filosofia de investimentos de Warren Buffett demonstra que investir bem depende principalmente de:

- compreensão do negócio
- visão de longo prazo
- disciplina financeira
- análise racional

Esses princípios continuam sendo uma das abordagens mais respeitadas do mercado financeiro.
